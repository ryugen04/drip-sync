データ削除リクエスト

DripSync に保存されたデータは、すべてお使いのデバイス内にのみ保存されています。外部サーバーにはデータを送信していないため、以下の方法でデータを削除できます。

方法1: アプリのデータを消去

アプリを残したままデータのみを削除します。

  1. デバイスの「設定」を開く
  2. 「アプリ」または「アプリと通知」を選択
  3. 「DripSync」を選択
  4. 「ストレージ」を選択
  5. 「データを消去」をタップ

方法2: アプリをアンインストール

アプリとすべてのデータを完全に削除します。

  1. デバイスの「設定」を開く
  2. 「アプリ」または「アプリと通知」を選択
  3. 「DripSync」を選択
  4. 「アンインストール」をタップ

Health Connect のデータ

Health Connect に同期したデータは、Health Connect アプリから個別に削除する必要があります。

  1. 「Health Connect」アプリを開く
  2. 「データと権限」を選択
  3. 「水分補給」を選択
  4. 削除したいデータを選択して削除
